#### 3.1.8 FAQ Section

**Section Title**: Frequently Asked Questions

**FAQ Items** (display in accordion or expandable list):

1. **What is the AI Detector API?**
   - Answer: The AI Detector API is a REST API that allows you to integrate AI content detection into your application. It analyzes text and provides probability scores indicating whether content was generated by AI models like ChatGPT, Gemini, Claude, or written by humans.

2. **What is an AI Detection API?**
   - Answer: An AI Detection API is a programmatic interface that enables developers to detect AI-generated content in their applications. It provides JSON responses with detailed analysis including AI probability scores, sentence-level detection, and confidence metrics.

3. **Does the API support ChatGPT detection?**
   - Answer: Yes, our API detects content generated by ChatGPT and other leading AI models including GPT-5.5, Gemini, Claude, DeepSeek, Llama, and Mistral. We continuously update our detection capabilities as new models are released.

4. **How do I authenticate API requests?**
   - Answer: API requests are authenticated using API keys. Include your API key in the Authorization header as a Bearer token. You can obtain your API key from your account dashboard after signing up.

5. **What are the API rate limits?**
   - Answer: Rate limits vary by plan. Free plan includes 100 requests per month, Pro plan includes 10,000 requests per month, and Enterprise plan offers unlimited requests. Rate limits reset monthly.

6. **What programming languages are supported?**
   - Answer: Our REST API works with any programming language that supports HTTP requests. We provide official SDKs for popular languages including Python, JavaScript, PHP, Ruby, and Java.

7. **How accurate is the AI detection?**
   - Answer: Our API achieves high accuracy rates across multiple AI models. Accuracy varies depending on content length and complexity. Longer texts generally provide more accurate results. We continuously improve our detection algorithms.

8. **Can I use the API for batch processing?**
   - Answer: Yes, the API supports batch detection allowing you to analyze multiple texts in a single request. This improves efficiency for high-volume use cases.

9. **What is included in the API response?**
   - Answer: API responses include AI probability score, human probability score, confidence score, detected AI model, and sentence-level analysis. Responses are returned in JSON format for easy parsing.

10. **Do you offer enterprise support?**
    - Answer: Yes, Enterprise plan includes dedicated account manager, custom integration support, SLA guarantees, and priority technical support. Contact our sales team to discuss enterprise requirements.
